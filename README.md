# PostKrypt
Haskell assignment for Programming Languages and Paradigms (JPP) classes on UW. 
Converts PostKrypt (stack-based language for describing vector graphic) to PostScript. Full description (in PL) in test/ZADANIE.md
src/ contains code submitted/required for the task, test/ materials provided by the professors including examples of input/output. 
## Part A
Lib.hs file with functions and data structures for keeping/modifying vector graphics.
## Part B
Parses said structures (type Picture) into valid .ps file.
## Part C
Parses PostKrypt into said structures.
